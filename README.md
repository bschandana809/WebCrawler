# web crawler

## Description
This project is a simple web crawler and search engine implemented using Python.

## Features
- Web crawling
- Inverted indexing
- TF-IDF ranking
- Simple HTML search interface

## Technologies Used
- Python
- HTML
- JSON

## Files
main.py – main program  
indexer.py – indexing pages  
worker.py – worker process  
producer.py – crawling producer  
index.html – search interface
